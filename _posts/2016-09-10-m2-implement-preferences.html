Sometimes, you will want to override the default function of Magento, you can do this by many ways in Magento 2, the Preference is one of the options.

Assume, you want to custom the function to display wishlist item.

In default, Magento will display wishlist with format __('%s item', $count).

I don't want to display 'item', an alternative I use 'product' for that string, I will use the preference to override the default.

Create a new module or use any module exists, I call it is 'Theme_Wishlist'.

I will implement the preference in di.xml file place in Theme_Wishlist/etc/ folder:

https://gist.github.com/dhduc/84ffe8d737d0c90fc081558df958a80d

In preference tag, the for param is the default class that I want to override by the class in type param. So,  I will create a class 'Theme\Wishlist\CustomerData\Wishlist' extends from 'Magento\Wishlist\CustomerData\Wishlist'.

https://gist.github.com/dhduc/85ddd11dc027e0cf2eb43152dee83a2f

Enable the module:

php bin/magento module:enable Theme_Wishlist

php bin/magento setup:upgrade

php bin/magento cache:clean

You can add some products to wishlist to see the result.
