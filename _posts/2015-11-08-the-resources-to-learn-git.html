GIT - Là một phần mềm quản lý mã nguồn theo hệ thống phân tán giúp cho việc phát triển phần mềm một cách dễ dàng với nhiều tính năng rất hữu ích.

Mọi người có thể học GIT từ nhiều nguồn tùy thuộc vào việc tìm kiếm với các lựa chọn riêng, dưới đây là một số gợi ý:

English:

  • https://git-scm.com/ - Đây là site chính thống về GIT bao gồm khá nhiều các thông tin liên quan đến GIT, nội dung trên GIT SCM khá là hàn lâm hướng tới việc hiểu sâu về các hoạt động của GIT một cách đầy đủ.
  • https://www.atlassian.com/git/tutorials/ - Site hướng dẫn về GIT của Atlassian, Atlassian là nhà cung cấp dịch vụ khá phổ biến về GIT là Bitbucket https://bitbucket.org/, nội dung site nghiêng về thực hành với các hướng dẫn và thảo luận hướng vào cách sử dụng các tính năng của GIT phục vụ quy trình phát triển phần mềm.
  • http://www.tutorialspoint.com/git/ - Nội dung site khá rõ ràng, đưa ra các hướng dẫn cụ thể về cách sử dụng GIT với các ví dụ thực tế và kết quả sau từng ví dụ.

Tiếng Việt:

  • https://git-scm.com/book/vi/v1 - Phần Documention trên GIT SCM được translate sang tiếng Việt, tuy nhiên phần nội dung tiếng Việt mới hoàn thành 3 chương đầu tiên.
  • http://vietjack.com/git/git_managing_branches.jsp - Đây là site được dịch lại từ nội dung GIT Learning trên http://www.tutorialspoint.com/git/

Practice:

  • https://try.github.io/levels/1/challenges/1 - Mọi người có thể vừa học vừa thực hành GIT ngay trên site này, với các hướng dẫn đi kèm theo từng lesson, phù hợp với những người mới bắt đầu.

Server Site:

  • https://github.com/ -  Mạng xã hội cho các developer,programmer, coder giúp bạn lưu trữ các GIT repositpory, public và chia sẻ với cộng đồng.
  • https://bitbucket.org/ - Site này có nhiều tùy chọn để quản lý các GIT repository với các tính năng private, team working.
